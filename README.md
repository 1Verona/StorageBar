# StorageBar

A tiny rounded progress bar that lives in your Mac's menu bar and shows how full your disk is at a glance. Pick a style, customize the color, and stop opening "About This Mac" every five minutes.

Made for people who watch their GBs.

## Build

Open `StorageBar.xcodeproj` in Xcode and press ⌘R. Requires macOS 13+ (developed against macOS 26).

## Distribution

Pre-built app available in `dist/StorageBar.app` (and `dist/StorageBar.zip`).

## License / Activation

24-hour trial on first launch. To unlock:
- Use a Gumroad license key, or
- Use the master key for free testing.

The Gumroad product ID is set in `GumroadConfig` inside `StorageBar/StorageBarApp.swift`.
